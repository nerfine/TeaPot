# TeaPot Bot - Privacy Policy

**Last Updated:** November 10, 2025

This Privacy Policy describes how TeaPot Bot ("we," "us," "our," or "the Bot") collects, uses, stores, and protects your information when you use our Discord bot service.

## 1. Information We Collect

### 1.1 Automatically Collected Data

When you use TeaPot Bot, we automatically collect:

**Discord User Data:**
- User ID (Discord's unique identifier)
- Username and discriminator
- Server (Guild) ID and name
- Channel IDs where commands are used
- Role IDs and names
- Message content when using Bot commands
- Avatar URLs
- Join dates and member counts

**Server Configuration Data:**
- Custom prefix settings (per-server and per-user)
- Welcome channel and message settings
- Moderation role assignments
- Log channel configurations
- Counting channel settings
- Game channel settings (Truth or Dare)
- Level role configurations

**Usage Data:**
- Commands used and their parameters
- Timestamps of interactions
- Error logs and debugging information

### 1.2 User-Provided Data

**Profile Claims:**
- Brawl Stars player tags you claim
- Discord-to-player tag associations

**Leveling System:**
- XP and level progression
- Message count per user
- Streak information
- Reward history

**Moderation Data:**
- Warning records (reason, moderator, timestamp)
- Moderation actions taken (kicks, bans, mutes)
- Audit log information

**Counting Game:**
- Current count per server
- User counting statistics
- Personal streaks and achievements
- Guild progress data

**AI Chat History:**
- Conversation history with DumbAI (temporary, per session)
- Truth or Dare questions and responses
- AI-generated content

**Giveaway Data:**
- Giveaway entries (user IDs)
- Giveaway configurations and winners
- Host information

**Battle Statistics:**
- Brawl Stars battle logs
- Player statistics and achievements
- Trophy changes and win/loss records

### 1.3 Third-Party Data

We integrate with external APIs that may collect additional data:
- **Brawl Stars API:** Player statistics and battle information
- **Hugging Face AI API:** Text inputs for AI generation (processed externally)

We use collected information for:

### 2.1 Core Functionality
- Processing commands and providing Bot features

### 2.2 Moderation and Safety
- Enforcing server rules and Bot terms

- Detecting and preventing abuse
- Maintaining server logs for administrators

### 2.3 Service Improvement
- Analyzing usage patterns to improve features
- Debugging errors and fixing bugs
- Monitoring Bot performance
- Developing new features

### 2.4 Communication
- Responding to support requests
- Sending service announcements
- Notifying about important changes

## 3. Data Storage and Security

### 3.1 Storage Methods

**Local Database:**
- SQLite database files stored securely
- JSON files for configuration data
- Encrypted where applicable

**Data Location:**
- Stored on secure servers
- Regular backups performed
- Access restricted to authorized personnel

### 3.2 Security Measures

We implement reasonable security measures including:
- Secure database access controls
- Regular security audits
- Encrypted data transmission
- Rate limiting to prevent abuse
- Input sanitization to prevent exploits

**However, no method of transmission or storage is 100% secure. We cannot guarantee absolute security.**

### 3.3 Data Retention

- **Active Data:** Retained while you actively use the Bot
- **Configuration Data:** Retained until manually deleted or Bot removal
- **Chat History:** Temporary (cleared after session or periodically)
- **Logs:** Retained for up to 90 days for debugging
- **Moderation Records:** Retained indefinitely unless manually deleted
- **Inactive Data:** May be deleted after 6 months of inactivity

## 5. Your Rights and Choices

### 5.1 Access and Control

You have the right to:
- **Access:** Request a copy of your data
- **Correction:** Request correction of inaccurate data
- **Deletion:** Request deletion of your data
- **Opt-Out:** Stop using the Bot to prevent further collection

### 5.2 Data Deletion

To delete your data:
1. Remove the Bot from your server (for server data)
2. Use the `/clearchat` command to clear AI chat history
3. Contact us to request complete data deletion
4. Use `/resetmyprefix` to reset your personal prefix

**Note:** Some data (moderation records) may be retained for legitimate interests.

### 5.3 Limiting Data Collection

You can limit data collection by:
- Not using certain features (AI chat, leveling, etc.)
- Configuring Bot features selectively
- Using commands minimally
- Not claiming external profiles (Brawl Stars)

## 6. Children's Privacy

If we become aware that we have collected information from a child under the minimum age without parental consent, we will take steps to delete that information promptly.
